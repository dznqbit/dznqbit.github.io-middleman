Stuffing some genius code inside <pre> tags?

Use a squiggly HEREDOC .

:ruby
  my_ruby_code = <<~'EOS'
    next_level = [1, 2, 3].map do |i|
      i + 1
    end

    puts next_level.join(" ")
  EOS
